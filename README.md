# GA Capstone Project

This repository contains all notebooks, presentations, and supporting documentation for the General Assembly capstone project, including both lightning talks and the final capstone modeling work.

## Folder Structure

```
ga-capstone/
├── lightning-talk-1/
│   ├── notebooks/        # Jupyter notebook(s) for lightning talk 1
│   └── presentation/     # Slides/deck for lightning talk 1
├── lightning-talk-2/
│   ├── notebooks/        # Jupyter notebook(s) for lightning talk 2
│   └── presentation/     # Slides/deck for lightning talk 2
├── capstone/
│   ├── notebooks/        # EDA, modeling, and fine-tuning notebooks
│   ├── data/              # Raw/processed datasets (or data-loading scripts)
│   ├── models/            # Saved model artifacts / checkpoints
│   └── src/               # Reusable python modules, training/eval scripts
└── README.md
```
